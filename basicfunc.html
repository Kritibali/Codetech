BASIC FUNCTIONS OF PYTHON :


  1. int function() :

    It converts a string or floating point number into integer.
    Eg : int(12.45)
        12


  2. float function() :

    It converts a string or an integer number into a floating point number.
    Eg : float(47)
        47.00


  3. str function() :

    It converts a number into string having double or single quotes.
    Eg : str(47)
        '47'


  4. print function() :

    It is to display the contents written inside paranthesis as it is.
    Eg : print("HTML and CSS unleashes our creativity.")
     HTML and CSS unleahes our creativity.


  5. type function() :

    It is to know the type of input.
    Eg : X="Hey!"
    type(X)
    <class 'str'>


  6. eval function() :

    The full form of eval is to evaluate. The eval function takes a string and returns the type of of input expected.
    Eg : >>> X=eval('123')
    >>>X
    >>>123
    Note : As we know, 123 is expected to be integer and if we input it as a string, it will evaluate it and result will be in integers.
    Now, if we want to know its type,it will be:
    >>>print(type(X))
    <class 'int'>
