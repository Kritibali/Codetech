Introduction to functions

With the help of functions an entire program can be divided into small independent modules. Each small module is called a function.
Syntax and basics : A function is a self contained block of one or more statements that performs a special task when called.
Syntax :
def name_of_function(Parameters)

Parameters and Arguments in function :

Parameters are used to give inputs to a function.They are specified with a pair of paranthesis.
Eg:printMax(10,20)----->Here printMax is the function name and 10,20 are parameters/arguments of function.

Positional Arguments :

The parameters are assigned by default according to their position.
Eg:def display(Name,Age)
  print("Name= ",Name,"Age= ",Age)
display("John",25)
display(40,"Rishab")------>#40 goes to Name and Rishab goes to Age
display("John")----->Error !!
Note:First argument binds to the first parameter and the second argument binds to second parameter. This style of matching up arguments and parameter is called 'positional argument style'

Keyword Arguments :

An alternative to postitional argument is keyword argument. This can be done by typing parameter_name=value in the function call.
Syntax:def display(Name,Age):
  print("Name= ",Name)
display(Age="24",Name="john")


Parameter with Default value :

We can provide default values to a parameter using assignment(=) operator.
Eg: def greet(name,msg="Welcome"):
  print("Hello",name,msg)
 greet("Sachin")
Output:Hello Sachin Welcome

The Local and Global scope of variable :

Variables that are initialised within a function including parameters are said to exist in function's local scope.
  • Variables that exist in local scope are called Local Variables.
  • Variables that are assigned outside functions are said to be Global Variables.


The 'return' statement :


The return statement is used to return a value from the function also break out from function.It can return multiple values.

Assign returned multiple values to variable :

It is also possible for a function to return multiple values and assign the returned multiple values to multiple variables.
Eg:def compute(num1):
  print("Number =",num1)
  return num1*num1,num1*num1*num1
square,cube=compute(4)
print("square=",square, "cube=",cube)------>Output : Number=4
  Square=16  Cube=64

Recursive functions :

A function is repetitively called by itself.A function is said to be recursive if a statement within the body of function calls itself.

The 'LAMBDA' function :

They are also known as anonymous functions. These functions are not bound to a name.They only have code to execute that is execute that is associated with them.
Syntax : Name=lambda(variable):code
Eg: def func(X) :
