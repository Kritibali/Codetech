INTRODUCTION TO OPERATORS

An operator indicates an operation to be performed on data to yield a result. It basically act upon operands which is the actual data.

INTRODUCTION TO EXPRESSIONS

An Expression is a block of code that produces a result upon evaluation.



    Arithmetic Operators

      Unary Operators      Binary Operators

      Unary Operators :

      Unary Operators are those which act upon only one operand. It includes ++(Increment Operator),--(Decrement Operator)..etc.

      Binary Operators :

      Binary Operators are those which act upon two operands. It includes +(Addition),-(subtraction),*(Multiplication),/(division),//(floor division),%(Modulo or Modulus Operator).**(Exponent Operator)


      Operator Precedence and Associativity

      Operator Precendence :

      Operator Precendence is all about priorities of operator. It tells about which operator should perform first for
      the accurate result in case we have multiple operators to be acted upon operands.


      Operator Associativity :

      It implies the direction of execution in case we have multiple operators of same precendence

    Bitwise Operator

    It permits a programmer to access and manipulate individual bits within a piece of data. There are 6 Bitwise Operators and here those are right below:

      & (Bitwise And)
      | (Bitwise OR)
      ^ (Bitwise XOR)
      >> (Right Shift)
      << (Left Shift)
      ~ (Bitwise NOT)

    Let's talk about these operators in detail :

      Bitwise AND (&) :

      It performs AND operation of given two values in bits and give the result in bits too.

      Bitwise OR (|) :

      Two operands are compared bit by bit and result will be in bits.

      Bitwise XOR (^) :

      Bitwise XOR gives high if both the inputs are different and it gives low if both inputs are high.

      Right Shift (>>) :

      It is to shift given number of bits to right.

      Left Shift (<<) :

      It is to shift given number of bits to left.

      Bitwise NOT:

      It compliments(reverse) the bits and result will be in bits. Calculate its decimal equivalent thereafter.



    Realtional Operator

    Relational operators are used to establish some sort of relationship between the two operands. Some of the relevant examples could be less than, greater than or equal to operators. Python language is capable of understanding these types of operators and accordingly return the output, which can be either True or False.
