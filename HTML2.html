TOKENS

Python breaks each statement into a sequence of lexical components known as TOKENS .

LITERALS

Literals are numbers or strings or characters that appear directly into a program. In other words, literals are constants.

Eg: 78 - Integer Literal 'W' - String Literal etc.

KEYWORDS

Keywords are reserved words with fixed meanings. There are 32 keywords in Python.


          

DELIMITER

Delimiters are symbols that perform grouping punctuations and assignment. Eg: {},[],+=,&= etc.

OPERATORS

Operator helps us to performs different tasks like addition, multiplication etc. Operators are applied on Operands . Operands are the data on which operators act upon. Eg: 4+3 ---> 4 and 3 are operands and '+' is operator.

There are different types of operators :

    Arithmetic Operator
    Relational Operator
    Logical Operator
    Bitwise Operator
    Boolean Operator
    Assignment/Compound Assignment Operator

IDENTIFIER/VARIABLE

Identifier is the name used to find a variable ,function,class etc. An Identifier is :

  • Is a sequence of characters that consists of letters, digits and underscores.
  • can be of any length.
  • can start with underscore(_).
  • cannot start with digit.
  • cannot be a keyword.
