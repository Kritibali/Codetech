Our programs are executed sequentially by default. Control flow refers to the order in which program statements are executed. We can change the control flow according to requirement.
Therefore, A programmer decides which statement is to be executed.

Introduction to Decision statements

BOOLEAN TYPE :

'Bool' has only two types viz. True and False.'Boolean' comes from the name of British Mathematecian, George Bool.
Note :Capitalisation of the first letter is important and so 'true' and 'false' are not considered as Boolean values.

BOOLEAN OPERATOR :

'AND' , 'OR' and 'NOT' are three basic boolean operators. The NOT operator has high precedence followed by 'AND' and then 'OR'.

  1. The NOT operator :

    2. It is a unary operator and compliment the given value.
    Eg : not True ------->False.

  2. The AND operator :

    3. It performs AND operation.
    Eg :True AND True ---------> True.

  3. The OR operator :

    4. It performs OR operation.
    Eg :True AND False ---------> True.

Decision Making Statements
  1. if statement
  2. if-else statement
  3. Nested if statements
  4. Multiway if-elif-else statement
  1. The 'if' statement :

    2. It executes if condition is true.
    Syntax : if condition :
        statement(s)

    Point to remember :
      The statement(s) must be intended atleast one space right of the if statement.


  2. The 'if-else' statement :

    3. It takes care of a true as well as false condition .
    Syntax :
     if condition :
        statement(s)
      else :
         statement(s)

    Point to remember :
      The indentation is must. The 'else' keyword must properly line up with 'if' statement . If not,it will show an 'identation error'

  3. The 'Nested if statements' :

    4. When one if statement is inside another if statement,then it is called a nested if statement.
    Syntax :
     if exp1 :
      if exp2 :
       statement 1
      else :
       statement 2
      statement 3




  4. The 'Multiway if-elif-else' statement :

    5. It comes into place where we have multiple if statements or cases to solve. Boolean expressions are checked from top to bottom. When true condition is found, the statement associated with it is executed and rest of conditional statements are skipped.If none of conditions are found true,else statement will execute.

    Syntax :
    if exp1:
      statement 1
    elif exp 2 :
      statement 2
    elif exp3 :
       statement 3
    - - - - - - - - -
    - - - - - - - -
    elif expN :
      statement N
    else
     statement(s)
